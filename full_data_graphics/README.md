# Full Data Graphics

This folder contains only the analysis graphics for the **Variant C** orchestration as an example of the comparison results.

The complete analysis data and graphs can be generated by following the step-by-step instructions provided in the main [README.md](../README.md).

Please note that the full comparison analysis files are not included in this folder. This measure is taken to prevent the exposure of any personal data that might be present in the raw extraction outputs. For more information regarding the dataset and its privacy policies, please refer to the dataset section in the main repository documentation.
